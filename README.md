## Required

- **Node.js**: Version 18 or higher
- **npm**: The version that comes with Node.js
- **Browser**: Chrome, Firefox, or any modern browser

## Check node version

```bash
node -v
```

## Installation

```bash
npm install
```

## Running the app

```bash
# Development Mode
$ npm start

# Watch Mode
$ npm run start:dev
```

## Build the app

```bash
$ npm run start
```
